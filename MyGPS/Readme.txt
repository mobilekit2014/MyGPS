MyGPS
 - GPS를 이용한 위치확인 어플리케이션
 - 처음 앱 실행시 단말기의 GPS가 켜져있는지 꺼져있는지 확인
 - 네트워크 위치확인은 코드상에 주석처리 해놨음


사용 방법

1. 위치 확인 버튼을 누른다.
2. 토스트로 위도와 경도값이 출력된다.


처음 앱 실행시, GPS 설정

1. 처음 앱 실행시 GPS가 꺼져있으면 경고창이 뜬다.
2. 아니오를 누르면 경고창이 닫힌다.
3. 예를 누르면 GPS 설정화면이 뜬다.


앱 실행 도중, GPS 설정

1. GPS 설정 확인을 누른다.
2. GPS가 켜져 있으면 토스트로 이미 켜져있다는 문구가 출력된다.
3. GPS가 꺼져 있으면 위와 같은 경고창이 뜬다.
4. 예를 누르면 GPS 설정화면이 뜬다.
